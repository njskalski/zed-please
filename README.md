This is a 100% vibecoded minimal plugin for Zed wiring https://please.build LSP with ZED.
I needed it and had no time to do it any other way. It works good enough.

The tree-sitter grammar was generated by Claude reading the golang code.

I use it with settings

```json

"file_types": {
  "Please Build": ["BUILD", "BUILD.plz", "*.build_defs", "*.build_defs.plz"],
},
"lsp": {
  "please-build": {
    "binary": {
      "path": "/home/andrzej/.please/build_langserver",
    },
  },
},

```

and with [this branch](https://github.com/thought-machine/please/pull/3494) where I implemented
"show all usages".
